[https://github.com/karanmahato44/console-toggle](https://github.com/karanmahato44/console-toggle)

Quickly comment/uncomment console statements in your JavaScript/TypeScript codebase.

![Demo](https://github.com/karanmahato44/console-toggle/raw/main/assets/demo.gif)

## Default keybinds (change to your liking)

- **Linux / Winbloat:** `Ctrl` + `Alt` + `C`
- **Mac:** `Cmd` + `Option` + `C`

**Command Palette:** `Toggle Console Statements`

## Settings

`console.error` and `console.warn` are ignored by default. You can enable them in settings.

![Settings](https://github.com/karanmahato44/console-toggle/raw/main/assets/settings.avif)
